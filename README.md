# Gerador de Tabela Verdade

**Desenvolvido por [Gabriel][l1]**

Programa que gera uma matriz em forma de tabela verdade.

### Instruções

> Recomendo Linux

> CUIDADO!

> Para um valor muito alto pode consumir muita memória

```
gcc tabela-verdade.c -lm
./a.out
```
